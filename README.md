# cnn-vs-mlp-bloodmnist-pytorch

This project explores supervised image classification on BloodMNIST (MedMNIST v2) using PyTorch.

Current status:
- Implemented a CNN pipeline with train/validation/test splits.
- Tested multiple setups (loss functions, optimizers, dropout, and early stopping).
- Recorded experimental results and comparisons in [results.md](results.md).

Next step:
- Implement and evaluate an MLP baseline under the same protocol to compare against the CNN.
